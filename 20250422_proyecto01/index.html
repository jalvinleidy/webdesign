CURSO DE DISEÑO WEB

HTML 5

HTML5 is the latest version of HyperText Markup Language (HTML), used for structuring and presenting content on the web. It builds upon previous versions by introducing new elements, attributes, and behaviors that enhance multimedia support, improve accessibility, and enable more interactive web applications without relying on external plugins

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation and layout of web pages.
It allows developers to define styles for HTML elements, such as colors, fonts, spacing, and positioning, ensuring a consistent and visually appealing design across different devices and screen sizes.

JavaScript

JavaScript is a versatile programming language primarily used for creating interactive and dynamic content on web pages. It is a client-side scripting language, meaning it runs directly in a user's web browser without requiring server interaction.

Its features include:

  • InteractividadPermite la manipulación de elementos en una página web en respuesta a eventos del usuario.
  • CompatibilidadFunciona en todos los navegadores modernos sin necesidad de instalación adicional.
  • Programación orientada a objetosSoporta la creación y manipulación de objetos para estructurar el código de manera eficiente.
  • Integración con HTML y CSSSe usa para mejorar la apariencia y funcionalidad de los sitios web.
  • Uso en el backendCon entornos como Node.js, JavaScript también se emplea en el desarrollo de servidores y aplicaciones web.
